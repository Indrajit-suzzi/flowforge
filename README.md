# 🚀 FlowForge

FlowForge is a reusable backend system with dynamic schema handling and secure authentication using JWT.

---

## 🧠 Features

- 🔐 JWT Authentication (Login/Register)
- 🔒 Password hashing using bcrypt
- 🛡️ Protected routes with middleware
- ⚙️ Dynamic API generation (generic controllers)
- 🐳 Docker-ready backend setup

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Docker
- JWT (jsonwebtoken)
